Baner (przesuwany tekst)

Lorem ipsum dolor sit amet.

<div class="title" style="left:300px;">Lorem ipsum dolor sit amet.</div>const title = document.querySelector('.title');

window.setInterval(() => {
    if(parseInt(title.style.left) < 10){
        title.style.left = '250px';
    }
    else{
        title.style.left = (parseInt(title.style.left) - 3) + 'px';
    }
}, 100);.title{
    position: relative;
    left: 0px;
}

Tekst przesuwa się z prawej strony w lewą, a następnie z powrotem wraca na prawą stronę. W HTML'u tworzymy diva z klasą title u ustawiamy wartość left na 300px, aby domyślnie div był przesunięty na prawą stronę. Następnie w JS wyszukujemy div title i przypisujemy go do zmiennej. Tworzymy funkcję window.setInterval i jako pierwszy argument podajemy funkcję anonimową, a jako drugi argument ilość sekund co jaką ma się powtarzać funkcja. Jeżeli wartość left div'a title jest mniejsza niż 10 przywracamy go na prawą stronę ekranu. Jeżeli nie to odejmujemy od wartości left 3 piksele. W CSS zmieniamy ustawiany position na relative, aby mieć możliwość swobodnej zmiany położenia diva, a następnie ustawiamy wartość left na 0px.

Baner (tekst pływający)

Lorem ipsum dolor sit amet.

<div class="title">Lorem ipsum dolor sit amet.</div>const title = document.querySelector('.title'); // element, który ma być przesuwany
const x = 1; // ilość znaków, które mają być przesunięte w lewo

window.setInterval(() => {
    let content = title.textContent; // pobieramy zawartość elementu
    title.textContent = content.substring(x) + content.substring(0,x); // przesuwamy zawartość o x znaków w lewo
}
, 100);

Tekst jest wyświetlany w ciągłej pętli. Tak jak w poprzednim przyładzie w HTML'u tworzym diva z klasa title. W JS'ie znajdujemy ten element i przypisujemy go do zmiennej. Następnie ustalamy zmienną x, czyli liczbę znaków o którą będziemy przesuwać element (w tym przypadku 1).Tworzymy funkcję window.setInterval i jako pierwszy argument podajemy funkcję anonimową, a jako drugi argument ilość sekund co jaką ma się powtarzać funkcja. Zawartość div'a title przypisujemy do zmiennej (content). Zawartości div'a przypisujemy dwa połączone ze sobą łańcuchy:

  • content.substring(x) - z zawartości .title ucinamy od x do końca.
  • content.substring(0,x) - z zawartości .title ucinamy od początku do końca.

Animowanie grafiki


<img class="image" src="img/image1.jpg" alt="Zdjęcie">const images = ['img/image1.jpg','img/image2.jpg']

const image = document.querySelector('.image');

image.addEventListener('mouseover', () => {
    image.src = images[1];
});
image.addEventListener('mouseout', () => {
    image.src = images[0];
});

Zdjęcie zmienia się po najechaniu po myszą, a następnie wraca do pierwotnego stanu. W HTML'u tworzymy img z klasą image i przypisujemy mu domyślne zdjęcie. W JS tworzymy listę z ścieżkami do zdjęć oraz wyszukujemy element utworzony w HTML. Korzystamy z addEventListener aby zarejestrować zdarzenie najechania i zjechania myszą ze zdjęcia (mouseover, mouseout). Po wywołaniu zdarzenia ustawiamy źródło zdjęcia (image.src) na jedną ze ścieżek dostępnych w stworzonej wcześniej liście images.

Sprawdzanie formularza


<form action="">
    <label for="name">Imię: <input type="text" id="name"></label>
    <label for="surname">Nazwisko: <input type="text" id="surname"></label>
    <label for="job">Zawód: <input type="text" id="job"></label>

    <input type="submit" value="Wyślij">
</form>const name = document.querySelector('#name');
const surname = document.querySelector('#surname');
const job = document.querySelector('#job');
const form = document.querySelector('form');

const checkForm = () => {
    if (name.value.length < 3) {
        alert('Pole Imię musi mieć przynajmniej 3 znaki');
        name.focus();
        return false;
    }
    if (surname.value.length < 3) {
        alert('Pole Nazwisko musi mieć przynajmniej 3 znaki');
        surname.focus();
        return false;
    }
    if (job.value.length == "") {
        alert('Pole Zawód musi być wypełnione');
        job.focus();
        return false;
    }
    alert('Formularz wypełniony poprawnie!');
    return true;
}

form.addEventListener('submit', function (event) {
    event.preventDefault();
    checkForm();
});

Po wypełnieniu i wysłaniu formularza sprawdzane zostają wartości w każdym z pól. W HTML'u tworzymy formularz składający się z 3 pól tekstowych, przycisku submit i nadejemy im identyfikatory. W JS wybieramy elementy formularza oraz sam formularz i przypisujemy je do zmiennych. Tworzymy funkcję strzałkową checkForm, która sprawdza czy wartości w elementach formularza mają określone długości. Jeżeli nie spełniają wymogów wysyłamy alert użytkownikowi, ustawiamy focus na tym elemencie i zwracamy fałsz. Na koniec zwracamy prawda i wysyłamy wiadomość, że formularz został wypełniony pomyślnie (ta część kodu nie zostanie wywołana jeżeli poprzednie warunki zostały spełnione). Następnie korzystamy z addEventListener (nazwa eventu to - submit), aby wywołać funkcję checkForm. Warto zwrócić uwagę na zastosowanie event.preventDefault() co zapobiega restartowaniu strony.

Wyrażenia regularne

const pattern = /^[0-9]+[a-z]+$/;

Są to wzorce opisujące łańcuch symboli. Przy ich użyciu można sprawdzać i modyfikować teksty. Wyrażenia regularne są wykorzystywane do sprawdzania, czy ciągi znaków wprowadzonych do formularza są zgodne z wymaganymi wzorcami. Jeżeli chcesz zobaczyć wyrażenia regularne w akcji kliknij tutaj.
